## 介绍

**Lerator ** *(/lerator/)* ，取自 `Leaderboard generator` 中的 `Le` 与  `rator` 。

该项目主要用于维护排行榜信息，生成排行榜图片。

现使用`Python`进行开发，以`Tkinter`实现图形界面，`Sqlite`作为数据库。

## 功能

+ 维护多个排行榜
  + 向排行榜插入新项目
  + 修改排行榜中已有项目
  + 删除排行榜中已有项目
  + Excel导入
  + Excel导出

+ 数据库维护
  + 数据库备份
  + 数据库恢复
+ 打印排行榜

## 使用

1. 下载[release](https://github.com/Ray-Keiyaku/Lerator/releases)中二进制程序使用。
2. 克隆仓库后手动编译使用。
